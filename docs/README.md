# Amethyst Docs

These docs explain the current Amethyst project direction, local usage, block behavior, development workflow, and legal attribution.

Amethyst is early. Prefer honest docs over polished promises. The web editor, `.amx` project flow, and Electron desktop build are all active now; some 3D workflows are still rough.

## Source Code And Desktop Builds

Amethyst's source code is free, and building from source is free. The paid desktop option is for the pre-built binary distributed by the project, not for access to the source code.

Desktop builds add release-only convenience features such as the single-file offline HTML exporter. The source remains available, and building from source remains free.
